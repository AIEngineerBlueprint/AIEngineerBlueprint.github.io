# AIEngineerBlueprint

AIEngineerBlueprint is a production-focused static HTML learning platform for AI engineers. It is designed for GitHub Pages at `https://AIEngineerBlueprint.github.io` and works without a backend.

## What is included

- 32 learning tracks and 500+ generated chapter pages.
- Static HTML, CSS, JavaScript, SVG assets, search index, sitemap, manifest, and service worker.
- Dark/light mode, instant command-palette search, LocalStorage progress, bookmarks, quizzes, reading time, breadcrumbs, previous/next navigation, keyboard shortcuts, and expandable SVG diagrams.
- Production-oriented lessons covering LLM internals, RAG, MCP, agents, guardrails, AI security, responsible AI, AI in SDLC, AWS, Azure, local AI, coding assistants, and interview preparation.

## Local development

```bash
node scripts/generate-site.js
npm run serve
```

Open `http://localhost:8080`.

## GitHub Pages deployment

1. Create a repository named `AIEngineerBlueprint.github.io`.
2. Copy or push this project to the repository root.
3. In GitHub, enable Pages from the default branch root.
4. Visit `https://AIEngineerBlueprint.github.io`.

## Content generation

The site is generated by `scripts/generate-site.js`. Edit the track and chapter arrays in that file, then rerun:

```bash
node scripts/generate-site.js
```

You can run on a different port with:

```bash
PORT=3000 npm run serve
```
