OpenAI Codex · Chapter 14

Codex lab

The Codex lab is a hands-on exercise to run one real, moderately complex task through Codex end to end, from a scoped prompt through test verification, then reflect on what sandbox, approval, or AGENTS.md configuration would improve the next run.

Calculating reading timeBuild project

What you're building, and why

In practice it comes down to picking one real repository task, writing a scoped AGENTS.md and prompt, running it with a chosen sandbox mode, verifying with tests, and writing down what needed human correction and why.

It is like a supervised trial run that ends with a debrief on what to configure differently next time, not just whether the task succeeded.

The brief

Run one real, non-trivial task through Codex including test verification, and write down what AGENTS.md rule or sandbox setting would have prevented the correction you had to make.

A worked reference

A lab might take a real open issue, write a project AGENTS.md with the test command and one constraint, run the task in workspace-write mode, verify with tests, and note one thing that needed manual correction.

Starting point in code

# Lab: run one real issue through the full loop
codex --sandbox workspace-write \
  "read issue #142, propose a plan, implement it, and run the test
  suite. Stop and ask before touching any file outside src/orders/."
# Afterward: what needed correction, and what AGENTS.md rule would
# have prevented it?

Change the inputs and watch which assumption breaks first. The gap between what you expected and what you got is where the learning is.

Trade-offs to document

  • The core tension: Using a real, messy task teaches more about the tool's actual limits than a clean toy example, but it takes longer and can surface unrelated repository issues that distract from the learning goal

Definition of done

Watch out for

  • The lab's goal is not a perfect diff on the first attempt; it's identifying exactly where you had to intervene and why, since that's the judgment production usage of an agentic tool actually requires.
